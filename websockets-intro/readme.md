### steps to make a basic websocket server with tsx
 - npm init -y
 - npm i typescript ( if not installed globally )
 - npx tsc --init
 - configure the - '**tsconfig.json's** '  items : 
 - "rootDir" : "./src"
 - "outDIr" : "./dist"
 - install ws : **npm i ws @types/ws**
 - install express : **npm i express @types/express**
