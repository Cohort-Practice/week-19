## Chat App
this app is a simple chat application 

**with the features :** 
users can chat along with people online in the server
 
**how can you use this in your device :**
 - clone the repository 
 - go on each folder " cd client " & "cd server" 
 - run on each " npm install "
 - for client run "npm run dev"
 - for server run "npm start"
 - and then you go ...
